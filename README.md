# Markdown Syntax

## Italics and Bold
>* _Italics_: * or _  
>* **Bold**: ** or __  
>* **_Italics and Bold_**: _** for legibility  
## Headers
>ex) Header4: ####  
## Links
> ### Inline link
>* brackets([]) and parenthesis(())  
>* [[Text]]((link address))  
>* emphasize using bold in text  
> ### Reference link
>* this is a [reference link][reference]  
>* "[reference]: address"  
## Images
>! in front; the rest is same as links  
## Blockquotes
>preface a line with >  
## Lists
> ### Unordered lists
>* asterisk (*) + space  
> ### Ordered lists 
>* number + space  
## Paragraphs
>* 2 spaces for new line  
>* no spaces needed for headers  
## Codes 
> use ~~~
